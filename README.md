# Projeto API de Aluguéis de Casa

## 📖 Sobre o Projeto

Este projeto consiste na construção de uma API REST para gerenciamento de aluguéis de casas, desenvolvida com foco em boas práticas de programação, organização de código, validações de dados e arquitetura escalável.

O objetivo é fornecer uma estrutura sólida para cadastro e gerenciamento de imóveis, locatários e contratos de aluguel, simulando um cenário real de mercado imobiliário.

---

##  Funcionalidades

- Cadastro de casas para aluguel
- Listagem de imóveis disponíveis
- Busca de imóveis por ID
- Atualização de informações dos imóveis
- Exclusão de imóveis
- Validação de dados de entrada
- Tratamento de erros
- Middleware para validações
- Organização em camadas (Controllers, Services, Routes, Middlewares)

---

## 🛠️ Tecnologias Utilizadas

- Node.js
- Express.js
- JavaScript
- Nodemon
- Insomnia/Postman
- Git e GitHub
